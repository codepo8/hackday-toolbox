The Hackday Toolbox

Version: 1.0 (28/07/2010) - check for updates on GitHub

The Hackday Toolbox is a collection of demo code that should get you going when you are competing in a hackathon or hackday. It was compiled to avoid having to explain the same problems over and over again and instead to give you a simple example to start from.

This is not meant to be a document to surf but instead to try out and look at the source code. All the client-side examples have all the code in-line and you can start altering it to your needs. The PHP examples you need to look at the source in the editor of your choice.

You can download the Hackday Toolbox at GitHub. All the code here is licensed with BSD - so go nuts.
